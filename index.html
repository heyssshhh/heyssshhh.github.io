Reallyread.it

Help Reallyread.it develop a mobile version of its Chrome extension (checkout their website reallyread.it). Goal of the app is to create a platform that has engaged readers and filter out uninformed voices. Tracks whether user had 'really read' the content, and only allows those who have fully read the content to comment on the article.

Record Atlas

Unlike other social media platforms, Record Atlas is a student project which allow posts to be linked based on content rather than chronological order. Allows for more compelling storytelling process.

Snap Judgement

Use IBM Watson to help police make better decisions under high pressure. A proof of concept project using Watson's visual recognition service coupled with Rasberry Pi, Snap Judgement detects facial emoiton to alert the user of potential threats.

DartCourses

Smoother course selection experience for Dartmouth students. Crawled various school websites to gather info students need when choosing classes. Presented in a centralized location. Mission Incomputable

A game which uses the pebble watch and involved 3 parts: the field agent, guide agent and game server. Field agents wear the watch and travel to locations to secure 'code drops'. Guide agents advise the field agents to the correct locations while avoiding other field agents so they won't be captured.
